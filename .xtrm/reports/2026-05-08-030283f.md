## Code Changes

### Pi runtime package model
- `cli/src/core/pi-runtime.ts` now centralizes xt-managed package inventory with `getXtManagedPiPackages()`, includes `PROJECT_EXTENSION_PACKAGE` for `npm:@jaggerxtrm/pi-extensions`, and exposes freshness contracts (`PiPackageFreshnessState`, `PiPackageVersionInfo`, `PiPackageVersionProvider`, `PiPackageFreshnessStatus`).
- `getManagedPiPackageFreshness()` is provider-injected and network-free; command layers decide how to obtain installed/latest versions.
- `ensureAlwaysGlobalPiPackages()` now assures every canonical xt-managed Pi package in global/user Pi scope instead of a hard-coded two-package allowlist.
- `getXtManagedPiPackageDoctorReport()` (doctor helper) reports current/missing/outdated/version-unknown states with remediation and timeout-bounded npm lookup.

### `xt update`
- `cli/src/commands/update.ts` wires `assureXtManagedPiPackages()` into dry-run/apply behavior.
- Dry-run reports missing/outdated global xt Pi packages; `--apply` refreshes only stale managed packages and preserves unrelated user packages by additive managed-package operations.
- Human help/output and JSON output now make package freshness behavior explicit.

### `xt doctor`
- `cli/src/commands/doctor.ts` now includes `piPackages` in JSON output and a text Pi packages section.
- Doctor reports missing/outdated/version-unknown global xt-managed Pi packages with remediation commands, but never installs or mutates state.
- Text output warns that package freshness may contact npm and treats offline/unknown latest versions as explicit warnings rather than silently passing.

### Tests
- Pulled `origin/main` pi-mcp-adapter addition into the same canonical managed package inventory; targeted tests should include the Pi runtime safeguards after merge.
- Added `cli/src/tests/doctor.test.ts` for all-present, missing, outdated, version-unknown, JSON parity, and no-mutation doctor behavior.
- Expanded `cli/src/tests/pi-runtime-safeguards.test.ts` for canonical inventory, freshness classification, global package assurance, and doctor helper states.
- Updated `cli/src/tests/update.test.ts` and `cli/test/pi-runtime.test.ts` to use canonical package inventory and deterministic JSON parsing.
