## Code Changes

### `xtrm-ui` compact/native tool rendering

- `packages/pi-extensions/extensions/xtrm-ui/index.ts` now clears native built-in tool active-call state inside the `tool_result` handler. This avoids the transient pending-call row that was visible before the final compact result row.
- External background chrome no longer applies a full-row background or extra spaces. It truncates to the available width from column zero and applies `highlightExternalToolBadge()` directly.
- External badge styling now matches `^(•\s+)(\S+)`, preserving the bullet unstyled and applying only a dark-on-cold background to the first displayed tool-name token. It intentionally avoids bold and avoids adding visible padding so it cannot reintroduce the width-overflow crash fixed earlier.
- `EXTERNAL_TOOL_FRAME_PATCH_VERSION` is now `11`, ensuring Pi `/reload` replaces old prototype wrappers.
